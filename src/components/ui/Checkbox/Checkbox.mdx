# Checkbox

The Checkbox component permits you to make a binary choice.

## Importing

```javascript
import { Checkbox } from 'amazon-chime-sdk-component-library-react';
```

## Example

<ThemeProvider theme={lightTheme}>
  <GlobalStyles />
  <Canvas>
    <Checkbox value="" checked onChange={() => console.log('change')} />
    <Checkbox value="" checked={false} onChange={() => console.log('change')} />
  </Canvas>
</ThemeProvider>

```jsx
<Checkbox value="" checked onChange={() => console.log('change')} />
```
