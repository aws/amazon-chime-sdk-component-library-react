# VideoGrid

The VideoGrid component adapts to screen size and orientation, ensuring consistency of [VideoTile](/?path=/docs/ui-components-video-videotile--video-tile)'s across layouts.


## Importing

```javascript
import { VideoGrid, VideoTile } from 'amazon-chime-sdk-component-library-react';
```
