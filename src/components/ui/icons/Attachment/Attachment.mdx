# Attachment Icon

The Attachment component creates a clip icon using inline SVG. Typically, you can use it to represent an attachment action.

## Importing

```javascript
import { Attachment } from 'amazon-chime-sdk-component-library-react';
```

## Example

With custom width and height

    <Attachment width="2rem" height="2rem" />
  </Preview>
</ThemeProvider>
