# GalleryLayout Icon

The GalleryLayout component creates an icon using inline SVG. Typically, you can use it to indicate the layout of `VideoTileGrid`.

## Importing

```javascript
import { GalleryLayout } from 'amazon-chime-sdk-component-library-react';
```
