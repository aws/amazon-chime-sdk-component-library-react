# Sound Icon

The Sound component creates a speaker icon using inline SVG. Typically, you can use it to indicate volume or status of attendee's audio output.

## Importing

```javascript
import { Sound } from 'amazon-chime-sdk-component-library-react';
```

## Example

With custom width and height

<ThemeProvider theme={lightTheme}>
  <Preview>
    <Sound width="2rem" height="2rem" />
  </Preview>
</ThemeProvider>

```jsx
<Sound width="2rem" height="2rem" />
```

Disabled sound icon

<ThemeProvider theme={lightTheme}>
  <Preview>
    <Sound width="2rem" height="2rem" disabled title="Sound off" />
  </Preview>
</ThemeProvider>

```jsx
<Sound width="2rem" height="2rem" disabled title="Sound off" />
```
