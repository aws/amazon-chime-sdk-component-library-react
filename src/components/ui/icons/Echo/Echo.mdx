# Echo Icon

The Echo component creates a echo icon using inline SVG. Typically, you can use it in a meeting roster to indicate status of attendees who join the meeting via echo.

## Importing

```javascript
import { Echo } from 'amazon-chime-sdk-component-library-react';
```

## Example

With custom width

<ThemeProvider theme={lightTheme}>
  <GlobalStyles />
  <Canvas>
    <Echo width="2rem" />
  </Canvas>
</ThemeProvider>

```jsx
<Echo width="2rem" />
```

Muted Echo icon

<ThemeProvider theme={lightTheme}>
  <Canvas>
    <Echo width="2rem" muted />
  </Canvas>
</ThemeProvider>

```jsx
<Echo width="2rem" muted />
```
