# Up and Down Caret Icon

The UpAndDownCaret component creates a up and down facing carets icon using inline SVG. Typically, you can use it to represent an more options action.

## Importing

```javascript
import { UpAndDownCaret } from 'amazon-chime-sdk-component-library-react';
```

## Example

With custom width and height

  </Preview>
</ThemeProvider>
