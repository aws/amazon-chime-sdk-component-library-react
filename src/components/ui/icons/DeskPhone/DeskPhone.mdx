# DeskPhone Icon

The DeskPhone component creates a phone icon using inline SVG. Typically, you can use it in a meeting roster to indicate status of attendees who join the meeting via phone.

## Importing

```javascript
import { DeskPhone } from 'amazon-chime-sdk-component-library-react';
```
