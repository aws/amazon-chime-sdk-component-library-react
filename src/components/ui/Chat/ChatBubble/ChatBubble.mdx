# ChatBubble

The ChatBubble component displays incoming and outgoing chat messages and data.
It also determines the style of the bubble including color, chat bubble tail visibility, and name visibility.

## Importing

```javascript
import {
  ChatBubble,
  MessageAttachment,
} from 'amazon-chime-sdk-component-library-react';
```
