# Chat Message

The Message component displays incoming and outgoing chat messages and data.

## Importing

```javascript
import { Message } from 'amazon-chime-sdk-component-library-react';
```

## Example

<Canvas style={{ 'background-color': '#eee' }}>
  <ThemeProvider theme={lightTheme}>
    <GlobalStyles/>
    <Message variant='outgoing' name='Me' time='9:45 AM' content='This is an outgoing message' />
    <Message variant='incoming' name='Sarah Anderson' time='9:47 AM' content='This is an incoming message. It is much longer. This amount of text reaches the max length and goes onto the next line.' />
  </ThemeProvider>
</Canvas>
