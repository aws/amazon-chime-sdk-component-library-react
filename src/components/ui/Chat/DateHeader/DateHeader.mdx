# DateHeader
 
A DateHeader is used to separate posts in feed, such as chat messages by date. 

## Importing

```javascript
import { DateHeader } from 'amazon-chime-sdk-component-library-react';
```

## Example

<Preview>
  <ThemeProvider theme={lightTheme}>
  <GlobalStyles/>
    <Flex css="background-color: #e4e9f2">
      <DateHeader date={new Date().toString()} />
    </Flex>
  </ThemeProvider>
</Preview>


```jsx
<DateHeader date={new Date().toString()} key='1' />
```
