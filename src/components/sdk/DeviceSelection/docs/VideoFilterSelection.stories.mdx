# BackgroundBlurCheckbox

The `BackgroundBlurCheckbox` component renders a `Checkbox FormField` to select a background blur device. You can use this along with the `PreviewVideo` to 
add background blur onto the currently selected video device.

The component depends on the `DevicesProvider`. If you are using `MeetingProvider`, `DevicesProvider` is rendered by default.

## Importing

```javascript
import { BackgroundBlurCheckbox } from 'amazon-chime-sdk-component-library-react';
```

## Usage

```jsx
import React from 'react';
import {
  BackgroundBlurProvider,
  MeetingProvider,
  BackgroundBlurCheckbox,
  PreviewVideo
} from 'amazon-chime-sdk-component-library-react';

const App = () => {
  return (
    <BackgroundBlurProvider >
      <MeetingProvider>
        <PreviewVideo />
        <BackgroundBlurCheckbox />
      </MeetingProvider>
    </BackgroundBlurProvider >
  );
};
```

- `BackgroundBlurProvider`
