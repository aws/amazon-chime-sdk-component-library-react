# VideoInputControl

The `VideoInputControl` component renders a `ControlBarButton` with pop over menu options to select through multiple video input sources.
These video input sources are provided by the `useVideoInputs` hook.

When you click the button, the local video input toggles between on/off state.

You can use it in the [ControlBar component](/?path=/docs/ui-components-controlbar--control-bar) to build the meeting controls bar.

## Importing

```javascript
import { VideoInputControl } from 'amazon-chime-sdk-component-library-react';
```

## Usage

```jsx
import React from 'react';
import {
  MeetingProvider,
  VideoInputControl
} from 'amazon-chime-sdk-component-library-react';

const App = () => {
  return (
    <MeetingProvider>
      <VideoInputControl />
    </MeetingProvider>
  );
};
```
