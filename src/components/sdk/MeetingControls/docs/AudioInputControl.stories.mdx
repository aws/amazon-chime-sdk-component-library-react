# AudioInputControl

The `AudioInputControl` component renders a `ControlBarButton` with pop over menu options to select through multiple audio input options.
These audio input options are provided by the `useAudioInputs` hook.

When you click the button, the local audio input toggles between muted/unmuted state.

You can use it in the [ControlBar component](/?path=/docs/ui-components-controlbar--control-bar) to build the meeting controls bar.

## Importing

```javascript
import { AudioInputControl } from 'amazon-chime-sdk-component-library-react';
```

## Usage

```jsx
import React from 'react';
import {
  MeetingProvider,
  AudioInputControl
} from 'amazon-chime-sdk-component-library-react';

const App = () => {
  return (
    <MeetingProvider>
      <AudioInputControl />
    </MeetingProvider>
  );
};
```
