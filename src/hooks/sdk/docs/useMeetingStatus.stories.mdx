# useMeetingStatus

Returns one of the status from below meeting status enum.

```javascript
enum MeetingStatus {
  Loading,
  Succeeded,
  Failed,
  Ended
}
```

### Return value

```javascript
meetingStatus: MeetingStatus
```

## Importing

```javascript
import { useMeetingStatus } from 'amazon-chime-sdk-component-library-react';
```

## Usage

The hook depends on the `MeetingProvider`.

```jsx
import React from 'react';
import { MeetingProvider, useMeetingStatus } from 'amazon-chime-sdk-component-library-react';

const App = () => (
  <MeetingProvider>
      <MyChild />
  </MeetingProvider>
);


const MyChild = () => {
  const meetingStatus = useMeetingStatus();

  return (
    <p>Meeting Status: {meetingStatus}</p>
  )
}
```

### Dependencies

- `MeetingProvider`
