# Themes

The Chime SDK Components Library provides **_light_** and **_dark_** themes. You can customize your application by applying a theme to all components.

In Storybook, you can switch between themes for each component in **Themes** tab in **Canvas** view. You can refer to the usage and some examples below.

## Usage

```jsx
import {
  lightTheme,
  darkTheme,
  GlobalStyles,
} from 'amazon-chime-sdk-component-library-react';
import { ThemeProvider } from 'styled-components';

<ThemeProvider theme={lightTheme}>
  <GlobalStyles />
  <MyApp />
</ThemeProvider>;
```

## Example

<ThemeProvider theme={lightTheme}>
  <GlobalStyles />
  <Preview>

      <Button label="Basic button" />

      <SecondaryButton label="This is a secondary button" />
    </Flex>
  </Preview>
</ThemeProvider>
