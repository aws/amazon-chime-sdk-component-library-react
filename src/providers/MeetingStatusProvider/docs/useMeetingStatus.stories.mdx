# useMeetingStatus

Returns the current meeting status defined by `MeetingStatus` enum and a function to update the current meeting status. 

`MeetingStatus` enum

```javascript
enum MeetingStatus {
  Loading,
  Succeeded,
  Failed,
  Ended
}
```

If you are using `MeetingProvider`, `MeetingStatusProvider` is rendered by default.


### Return value

```javascript
{
  // Meeting status as defined by MeetingStatus enum
  meetingStatus: MeetingStatus;

  // A function to update the meeting status
  updateMeetingStatus: (status: MeetingStatus) => void;
}
```

## Importing

```javascript
import { useMeetingStatus } from 'amazon-chime-sdk-component-library-react';
```

## Usage

The hook depends on the `MeetingProvider` being rendered.

```jsx
import React from 'react';
import { MeetingProvider, MeetingStatus } from 'amazon-chime-sdk-component-library-react';

const App = () => (
  <MeetingProvider>
    <MyChild />
  </MeetingProvider>
);

const MyChild = () => {
  const { meetingStatus, updateMeetingStatus } = useMeetingStatus();

  const endMeeting = () => {
    updateMeetingStatus(MeetingStatus.Ended);
  }

  return (
    <div>
      <p>Meeting Status: {meetingStatus} </p>
      <button onClick={endMeeting}>End Meeting</button>
    </div>
  );
}
```

### Dependencies
- `MeetingProvider`
