# MeetingStatusProvider

Provides the current meeting status defined by `MeetingStatus` enum and a function to update the current meeting status. 

### `MeetingStatus` enum

```javascript
enum MeetingStatus {
  Loading,
  Succeeded,
  Failed,
  Ended
}
```

## State

```javascript
{
  // Meeting status as defined by MeetingStatus enum
  meetingStatus: MeetingStatus;

  // A function to update the meeting status
  updateMeetingStatus: (status: MeetingStatus) => void;
}
```

You can access the provided values using [useMeetingStatus](/?path=/docs/sdk-hooks-usemeetingstatus--page) hook.

## Usage

If you are using `MeetingProvider`, `MeetingStatusProvider` is rendered by default.

```jsx
import React from 'react';
import { MeetingProvider, MeetingStatus, useMeetingStatus } from 'amazon-chime-sdk-component-library-react';

const App = () => (
  <MeetingProvider>
    <MyChild />
  </MeetingProvider>
);

const MyChild = () => {
  const { meetingStatus, updateMeetingStatus } = useMeetingStatus();

  const endMeeting = () => {
    updateMeetingStatus(MeetingStatus.Ended);
  }

  return (
    <div>
      <p>Meeting Status: {meetingStatus} </p>
      <button onClick={endMeeting}>End Meeting</button>
    </div>
  );
}
```
