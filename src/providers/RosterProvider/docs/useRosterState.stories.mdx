# useRosterState

Returns state around the meeting's present attendees, also referred to as Roster. 

You must provide the `MeetingManager` with a `getAttendee` function in order to get names in your roster. See the [MeetingManager docs](?path=/docs/providers-meetingmanager--page) for more information.

### Return value

```javascript
{
 [AttendeeId: string]: {
   chimeAttendeeId: string;
   externalUserId?: string;
   name?: string;
 }
}
```

## Importing

```javascript
import { useRosterState } from 'amazon-chime-sdk-component-library-react';
```

## Usage

The hook depends on the `MeetingProvider` being rendered.

```jsx
import React from 'react';
import { MeetingProvider, useRosterState, Roster, RosterGroup, RosterAttendee } from 'amazon-chime-sdk-component-library-react';

const App = () => (
  <MeetingProvider>
      <MyChild />
  </MeetingProvider>
);

const MyChild = () => {
  const { roster } = useRosterState();
  const rosterArray = Object.values(roster);

    const { chimeAttendeeId, name } = attendee;
    return <RosterAttendee key={chimeAttendeeId} attendeeId={chimeAttendeeId} />;
  });

  return (
    <Roster>
      <RosterGroup>{attendeeItems}</RosterGroup>
    </Roster>
  )
}
```
