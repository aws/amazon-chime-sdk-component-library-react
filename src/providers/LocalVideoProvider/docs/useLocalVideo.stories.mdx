# useLocalVideo

The `useLocalVideo` hook returns the video state of the local user, and a function to toggle their video.

### Return Value

```javascript
{
  // The tile ID associated with the local user's video
  tileId: null | number;

  // Whether or not the local user has video enabled
  isVideoEnabled: boolean;

  // A function that toggles the local user's video
  toggleVideo: () => Promise<void>;
}
```

## Importing

```javascript
import { useLocalVideo } from 'amazon-chime-sdk-component-library-react';
```

## Usage

The hook depends on the `MeetingProvider` being rendered.

```jsx
import React from 'react';
import {
  MeetingProvider,
  useLocalVideo
} from 'amazon-chime-sdk-component-library-react';

const App = () => (
  <MeetingProvider>
    <MyChild />
  </MeetingProvider>
);

const MyChild = () => {
  const { isVideoEnabled, toggleVideo } = useLocalVideo();

  return (
    <button onClick={toggleVideo}>
      {isVideoEnabled ? 'Stop your video' : 'Start your video'}
    </button>
  );
};
```

### Dependencies

- `MeetingProvider`
